2.1筋トレ入門


 筋トレの始め方

  筋トレを始めたくても始められない。これにはそれぞれ理
  由があるかもしれないが、始め方がわからない人が多印象
  がある。ここでは筋トレが始めやすくなるよう私の意見を
  記していこう。




1. とりあえずやれ

  筋トレは家でやるかジムでやるか迷ってしまう。ジムに行
  きたいけど恥ずかしい。そんなことを考えていても筋トレ
  は始まらない。考えるよりも先に行動するべし。


2. 仲間を作ろう

  一人では続かなくても、友達などと複数人でやると。誰か
  一人でもやる気があれば筋トレできるので、自然と続くよ
  うになる。。


3. 習慣化しよう

  友達とかと続けていると、自然と筋トレが習慣化してくる
  ここまで来ればあとは簡単。ただの習慣として筋トレがで
  切るようになる。


4. モチベをあげよう

  習慣化してもしばらくすると飽きてきてしまう。そんな時
  に大事なのは、モチベの維持だ。自分の目標の再確認や理
  想の人、筋トレ動画を見てモチベをあげよう。




2.2食事内容


 食事の重要性

  筋肥大において食事は筋トレと同じくらい需要である。食
  事が疎かになってしまうと、いくら筋トレをしても効果が
  あまり期待できない。




1. 炭水化物

  体を動かすエネルギーの元となる栄養素。炭水化物が不足
  すると筋トレ中パワーが出なかったりだるさが出たりする
  一方で摂取しすぎると。体重増加に影響がでる。


2. 脂質

  脂質は悪だと考える人がいるかもしれないが、重要なエネ
  ギー源であり、脂質から合成される脂肪酸は疲労回復に大
  きく関わるため適正量をしっかりと摂取すべきである。

※魚類、オリーブ、ココナッツの脂質が良いとされている。


3. タンパク質

  タンパク質は筋肉のもととなる非常に重要な栄養素の一つ
  である。理想の摂取量は「体重(kg) × 2g」。
  


4. ビタミン

  タンパク質の吸収や筋合成を手助けしてくれる栄養素。緑
  黄色野菜に多く含まれる。また、豚肉に含まれるビタミン
  Bは疲労回復に効果がある。




2.3筋トレ用具


 用具がすべてなわけではない

  筋トレにおいて用具がなくてはならない。というわけでは
  ないが、よりけがの防止や一点により効かせたい時などに
  使用する。




1. リストラップ

  手首を守る道具で、主にベンチプレスやレイズ系の種目を
  行うときに使用する。
  


2. パワーグリップ

  前腕の疲労を軽減する道具で、主にプル系の種目を行うと
  きに使用する。前腕の疲労を抑えられるため、狙った部位
  により効かせられる。


3. パワーベルト

  腰を保護する道具で主にベンチプレス、デッドリフトなど
  を行うときに使用する。高重量を扱うときには付けたほう
  がいいと個人的に思う。


4. シェイカー

  プロテインを飲む上では欠かせないマストアイテム。筋ト
  レ初心者はシェイカーだけ持っていれば。十二分だ。ちな
  みに私はシェイカー以外の道具は持っていない。
